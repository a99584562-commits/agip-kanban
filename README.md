# АГИП · Прототип отчётов

Демо-стенд трёх отчётов по мероприятиям в виде канбан-досок:

1. **По спикерам** — колонки = спикеры
2. **По приоритетам** — колонки = Глава / Премьер и зампреды / Фон
3. **По целям** — колонки = Треки + Отработка рисков + Нац проект

Один и тот же набор мероприятий — три разных разреза. Слева единый фильтр.

## Локально

```bash
npm install
npm run dev
```

## Деплой

Автоматически на GitHub Pages при push в `main` (см. `.github/workflows/deploy.yml`).

## Доступ

Демо закрыто паролем (см. `src/components/Gate.tsx`). Данные тестовые,
сгенерированы для демонстрации структуры отчётов.
